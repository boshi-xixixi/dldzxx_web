# 项目技术栈

本文档详细介绍了企业网络监控数据看板系统所使用的核心技术、框架和库。

## 1. 前端 (Frontend)

| 技术分类 | 主要技术 | 描述 |
| --- | --- | --- |
| **核心框架** | [Vue 3](https://vuejs.org/) | 用于构建用户界面的渐进式 JavaScript 框架，利用其组合式 API (Composition API) 来组织和复用代码。 |
| **UI 组件库** | [TDesign](https://tdesign.tencent.com/) | 腾讯出品的企业级设计体系和 UI 组件库，提供了丰富、高质量的组件，保证了界面的一致性和专业性。 |
| **数据可视化** | [ECharts](https://echarts.apache.org/) | 一个强大的数据可视化库，用于在数据看板中创建动态、交互式的图表，如流量趋势图。 |
| **路由管理** | [Vue Router](https://router.vuejs.org/) | Vue.js 的官方路由管理器，用于实现单页面应用 (SPA) 的导航。 |
| **构建工具** | [Vite](https://vitejs.dev/) | 新一代前端构建工具，提供了极速的冷启动和模块热更新 (HMR)，显著提升了开发体验。 |
| **CSS 框架** | [Tailwind CSS](https://tailwindcss.com/) | 一个功能类优先的 CSS 框架，用于快速构建自定义的用户界面，与 TDesign 结合使用，实现了高效的样式定制。 |
| **图标库** | [Lucide](https://lucide.dev/) | 一个美观、一致的开源图标库，通过 `lucide-vue-next` 在项目中使用。 |
| **语言** | [TypeScript](https://www.typescriptlang.org/) | JavaScript 的超集，为项目提供了静态类型检查，增强了代码的健壮性和可维护性。 |

## 2. 后端 (Backend)

| 技术分类 | 主要技术 | 描述 |
| --- | --- | --- |
| **核心框架** | [Express.js](https://expressjs.com/) | 基于 Node.js 平台的极简、灵活的 Web 应用开发框架，用于快速搭建 RESTful API。 |
| **运行环境** | [Node.js](https://nodejs.org/) | 一个基于 Chrome V8 引擎的 JavaScript 运行时，使得我们可以使用 JavaScript 来编写后端代码。 |
| **TypeScript 支持** | [tsx](https://github.com/esbuild-kit/tsx) | 一个零配置的 TypeScript 执行器，可以直接运行 TypeScript 代码而无需预先编译，简化了开发流程。 |
| **开发工具** | [Nodemon](https://nodemon.io/) | 一个用于在开发过程中自动重启 Node.js 应用的工具，当检测到文件变更时，它会自动刷新服务器。 |

## 3. 开发与构建

| 技术分类 | 主要技术 | 描述 |
| --- | --- | --- |
| **包管理器** | [pnpm](https://pnpm.io/) | 快速、节省磁盘空间的包管理器，通过符号链接的方式管理 `node_modules`，避免了重复安装。 |
| **并发执行** | [Concurrently](https://github.com/open-cli/concurrently) | 一个可以同时运行多个命令的工具，在本项目中用于同时启动前端 Vite 开发服务器和后端 Express 服务器。 |
| **代码规范** | [ESLint](https://eslint.org/) / [Prettier](https://prettier.io/) | 用于保证代码风格和质量的工具，ESLint 负责代码质量检查，Prettier 负责代码格式化。 |

## 4. 部署 (Deployment)

| 技术分类 | 主要技术 | 描述 |
| --- | --- | --- |
| **平台** | [Vercel](https://vercel.com/) | 一个为前端开发者设计的云平台，提供了零配置的部署体验，支持自动化的构建和部署流程。 |
| **配置文件** | `vercel.json` | Vercel 的部署配置文件，用于定义路由重写、环境变量等，确保前后端路由正确映射。 |
